Aide mémoire des sélecteurs CSS déja vus

  • * séléctionne TOUS les éléments de la page html
  • .ma-jolie-nav selectionne tous les éléments ayant le nom de class "ma-jolie-nav"
  • #mon-unique-id sélectionne tous les élements ayant le nom de mon id "mon-unique-id"
  • div, article, aside sélectionne toutes les balises <div>, <article> ET <aside>
  • nav a sélectionne les liens <a> inclus/imbriqués dans la balise <nav>
  • nav a:hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris

Propriété CSS utilisé dans cette page

Décrivez leur usage et leurs valeurs possibles
  1. background: couleur de fond de l'élement
  2. color: couleur de la police
  3. float: effet flottant
  4. font-family: police
  5. font-size: taille de la police
  6. height: hauteur de l'élément
  7. left: position à partir de la gauche de la fenêtre
  8. line-height: hauteur de l'interligne
  9. margin: marges extérieures
  10. padding; marges intérieures
  11. position: position de l'élément dans la page (on n'a utilisé que la valeur (fixed) pour l'instant)
  12. text-align: alignement du texte
  13. text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé)
  14. top: position à partir du haut de la fenêtre
  15. tansition: temps de transition lors d'une animation (ici sur les liens de la barre de menus)
  16. width: largeur de lélément par rapport à la taille de la fenêtre si on indique une valeur relative (%, em, ...)

Propriétés CSS à rechercher

Recherchez les propriétés ci-dessous, faites des tests avec sur cette page depuis la console de débuggage pour tester les valeurs qu'elles peuvent prendre
  1. font-weight: permet de mettre du gras ou non sur la police (bold,900) etc..
  2. letter-spacing: permet de définir l'écart entre les lettres (px,.rem) etc..
  3. word-spacing: permet de définir l'écart entre les mots (px,rem,%) etc..
  4. text-shadow: permet de mettre une ombre sur le texte (px,rgb,hexadecimal) etc..
  5. background-image: permet de mettre une ou plusieurs images en arriere plan (url,gradient) etc..
  6. background-size: permet de definir la taille des images en arriere plan (contain,no-repeat) etc..
  7. opacity: définit la transparence d'un élement (0.33,1,0) etc..
  8. border-style: permet de definir un style de bordure (none,dotted,solid) etc..
  9. box-shadow: permet de mettre une ombre sur un élement (px,rgb) etc..
  10. list-style-image: permet de mettre un logo/image a mes puces (inherit,initial) etc..
  11. border: permet d'ajouter une bordure etc..
  12. border-radius: permet de définir des coins arrondis pour une bordure d'un element (px,%) etc..
  13. border-width: Définit la largeur d'une bordure (em,px,thick) etc..
  14. font-variant: change la police pour l'element actuel (small,caps,slashed-zero) etc..
  15. background-color: permet de definir une couleur de fond (rgb,rgba,red) etc..
  16. !important: permet de spécifier que la propriété est la plus importante dans votre feuille CSS etc..
